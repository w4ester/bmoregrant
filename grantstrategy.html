Solution & Innovation

The Platform: 7 Interactive Modules

🎓

1. AI Learning Hub

Gamified curriculum with progress tracking. Complete challenges like writing effective prompts and building resume tools.

🧪

2. Prompt Lab

A/B testing interface to see how different prompts produce different results. Learn through iteration.

🛠️

3. Tool Builder

No-code interface to create custom AI tools. Example: "Build a tool that checks if I'm eligible for SNAP benefits."

📄

4. RAG Explorer

Document analysis. Upload resume, transcript, or benefits letter → AI extracts key info.

🔬

5. Model Lab

Compare AI models side-by-side. Test different parameters to understand AI behavior.

🔌

6. MCP Studio

Live tool testing. Call real APIs (job boards, benefits sites) through AI agents.

📊

7. Dashboard

Track progress, show portfolio of created tools, recommend next steps.

Technical Infrastructure

Hardware

  • Mac Studio M3 Ultra (384GB RAM, 60-core GPU) - $7,000
  • Serves 20 simultaneous users in workshop setting
  • Runs GPT-OSS-20B (via Unsloth GGUF) at 40 tokens/sec

Models (all open-source)

  • GPT-OSS 20B: Primary teaching model
  • GPT-OSS 120B: Advanced projects
  • DeepSeek R1: Math and reasoning tasks
  • Qwen3: Multilingual support (Spanish, Haitian Creole)

Fine-Tuning with Unsloth.ai

  • Baltimore-specific datasets (job postings, benefit forms, training programs)
  • Economic opportunity scenarios (resume writing, interview prep)
  • 2x faster training, 70% less VRAM vs. standard methods
  • Entire fine-tuning cycle: <4 hours on M3 Ultra
