## GitHub Pages Setup

If you haven't enabled GitHub Pages yet:

1. Go to your repository Settings
2. Scroll to "Pages" section
3. Under "Source", select "main" branch
4. Click "Save"
5. Your site will be live at: https://w4ester.github.io/bmoregrant/
